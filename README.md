# NodeSchoolSA
Our goal is to organise a Node.Js, io.js and MongoDB workshop in South Africa

## Topics
<ul>
 <li>Node.JS</li>
 <li>io.js</li>
 <li>Socket.io</li>
 <li>Express (Web Framework)</li>
 <li>Sails (REST Framework)</li>
 <li>MongoDB (Document orientated database with dynamic schemas)</li>
</ul>
